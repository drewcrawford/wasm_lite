This project is a "smol"-style rewrite of wasm-bindgen.

A checkout of wasm-bindgen is available in the 'wasm-bindgen' folder.

Design goals:

* Runner for major web browsers.
* Support with and without +atomics
* Unit test support
* Bind JS APIs to rust and vice versa
* Doctest support
* Simple, clean architecture
* Avoid dependencies

nice to have:
* interop with wasm-bindgen crates
